# Sistema de Veículos em Java

Projeto desenvolvido para praticar Programação Orientada a Objetos (POO) em Java.

## Funcionalidades
- Criação de classes de veículos
- Herança (Carro e Moto herdando de Veículo)
- Encapsulamento com getters e setters

## Tecnologias
- Java

## Objetivo
Aplicar conceitos de orientação a objetos na prática.
